# 🌿 Céu Irmão Lakota — Site Institucional

Site institucional desenvolvido para o **Espaço Céu Irmão Lakota**, localizado em **Mogi das Cruzes – SP**, dedicado a vivências espirituais, autoconhecimento e reconexão com saberes ancestrais.

O site apresenta informações sobre o espaço, serviços oferecidos, agenda de vivências e formas de contato.

WebMail : https://webmail.iron.acessoprotegido.com.br/?_task=mail&_mbox=INBOX.Drafts

# 📍 Sobre o projeto

Este projeto consiste em uma **landing page moderna com efeito parallax**, contendo:

* Slider com imagens em tela cheia
* Navegação por overlays (modais)
* Sessões institucionais
* Apresentação de serviços
* Agenda de eventos
* Formulário de contato
* Integração com WhatsApp
* Mapa de localização
* Links para redes sociais

O objetivo é proporcionar uma **experiência visual imersiva e intuitiva** para os visitantes.

---

# 🧰 Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**
* **Font Awesome**
* **Google Fonts (Poppins)**
* **Parallax UI**
* **Google Maps Embed**
* **ASP (Classic ASP)** para envio de formulário de contato

# ⚙️ Funcionalidades

### 🖼 Slider Parallax

* Apresentação visual com múltiplas imagens
* Controle por setas e pontos
* Tempo configurável

### 📅 Agenda

Carregamento dinâmico de eventos e vivências.

### 🧘 Serviços

Apresentação dos serviços oferecidos:

* Kambo
* Barras Access

Com botão direto para **WhatsApp**.

### 📩 Formulário de contato

Envio de mensagens através de **sendmail.asp**.

Campos:

* Nome
* Email
* Assunto
* Mensagem

### 📍 Mapa

Localização integrada via **Google Maps**.
